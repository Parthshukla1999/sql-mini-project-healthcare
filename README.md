#🏥 Healthcare Database Project
This project is a mini SQL practice project where I designed and queried a simple healthcare database using PostgreSQL. The goal was to simulate a hospital environment and strengthen SQL skills by working with realistic queries and scenarios.
#📌 Project Overview
Designed a schema with Patients, doctor, Appointments and payment tables.
Inserted sample data to simulate hospital visits and diagnoses.
Wrote queries to answer real-world data problems.
